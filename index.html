Sinnesorgane · QR-Level

Die Felder werden nacheinander aktiv. Auf einem aktiven Feld kann ein beliebiger noch nicht verwendeter QR-Code gescannt werden. Das gescannte Sinnesorgan wird dann diesem Level zugeordnet.

Spielstand

0 von 5 Level abgeschlossen

Nach dem Scannen öffnet sich eine kurze Abfrage. Starte danach das Level, bearbeite die Seite und klicke unten auf „Level fertig“. Danach wird das nächste Feld auf dem Spielbrett aktiv.

Legende

AktivDieses Feld kann angeklickt und mit einem beliebigen QR-Code freigeschaltet werden.
GesperrtDieses Feld wird erst nach dem vorherigen Level aktiv.
FertigDas Level wurde bearbeitet und abgeschlossen.
